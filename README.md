# Anki Landing Page Refresh

To build and view locally:

```sh
npm install
npm run build
npm run preview
```

## Or if you are living in 2024

```sh
bun install
bun run build
bun run preview
```

## Alternatively you can just do:

```sh
make preview
```

### Or

```sh
just preview
```

---

Then navigate to the provided URL.
